À propos du projet

Le projet

Ce Pokédex est un projet pédagogique réalisé dans le cadre du titre RNCP Développeur Web Front-End — Bloc 2, à La Plateforme.

Il affiche les 151 Pokémon de la première génération en consommant l'API publique PokéAPI, sans clé d'authentification requise.

Technologies utilisées

  • 📄 HTML5 sémantique (WCAG AA)
  • 🎨 SASS / SCSS compilé
  • 📱 Mobile-first + Flexbox / Grid
  • ⚡ JavaScript ES6+ (fetch, async/await)
  • 🌐 PokéAPI (API REST publique)
  • ✍️ Google Fonts — Silkscreen + DM Sans

Compétences démontrées

Découpage maquette

Analyse et traduction d'une maquette en blocs sémantiques HTML.

Accessibilité WCAG AA

Attributs aria, navigation clavier, contrastes suffisants, lang, alt.

SASS / Préprocesseur

Variables, nesting BEM, mixins, compilation automatique.

Responsive Mobile-first

Media queries ordonnées du mobile vers le desktop.

Communication asynchrone

Fetch + async/await vers PokéAPI, gestion des erreurs.

Manipulation DOM

Sélecteurs, événements, timers, création dynamique de contenu.
